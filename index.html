📖 How to Use CVForge Pro

Build an ATS-optimized, recruiter-ready CV in under 20 minutes using Groq AI.

🚀 Quick Start (5 steps)

1️⃣Get a free Groq API key at console.groq.com → click "🔑 Set Groq Key" in the top-right corner
2️⃣Enter your Target Job Title on Panel 1 and paste the full Job Description → click "Extract Keywords"
3️⃣Fill Personal Info → Summary → Experience → Skills across all panels. Use the AI buttons to assist each section.
4️⃣Go to Generate & Export (last panel), click "Generate with Groq" — takes 10–20 seconds
5️⃣Review your CV, click "Edit CV" to tweak inline, then export as PDF or RTF to send

🤖 AI Features Explained

✦ Extract Keywords

Reads your job description and extracts exact ATS keyword phrases. Use these to populate your Skills section.

✦ Generate Summary

Groq writes a keyword-dense professional summary tailored to your role. Always review and edit to sound authentic.

✦ AI Enhance Bullets

Transforms vague responsibilities into powerful ATS bullets with action verbs and metrics.

✦ AI Polish

After generating, AI reviews the full CV and strengthens weak verbs, adds specificity, and fixes formatting.

📤 Which Export Format to Use

📄PDF — Use for 90% of applications. Opens print dialog — set "Save as PDF".
📝RTF / Word — When employer requests Word format. Opens in Word, Google Docs, and LibreOffice.
📋TXT — For pasting into online application form text boxes. Maximum ATS compatibility.
🌐HTML — Download, open in browser, Ctrl+P → "Save as PDF" for maximum layout control.

💡 ATS Optimization Tips

🎯Paste the specific job description for each application — not generic.
📝Use exact keyword strings: "React.js" not "React", "Microsoft Azure" not "Azure cloud".
📊Quantify everything: "Managed team" → "Managed team of 12 engineers across 3 time zones".
🚫Never use tables, columns, or graphics in your ATS submission — they confuse parsers.
🔄Customize for each role: change Target Role + paste new Job Description + click Regenerate.
