# CVForge Pro — AI-Powered ATS CV Builder

Build ATS-optimized, recruiter-ready CVs in minutes using Groq AI.

## Features

- **AI Keyword Extraction** — Paste a job description, extract exact ATS keywords
- **AI Summary Generation** — Keyword-dense professional summaries tailored to your role
- **AI Bullet Enhancement** — Transform vague responsibilities into quantified action-verb bullets
- **AI Polish** — Full CV review and improvement after generation
- **ATS Score** — AI-powered scoring with keyword gap analysis
- **Multi-format Export** — PDF, RTF (Word), TXT, HTML
- **Auto-save** — All data persists in localStorage
- **Mobile-responsive** — Full mobile navigation with sidebar toggle
- **Accessibility** — ARIA labels, keyboard navigation, reduced-motion support

## Setup

1. Clone or download the repository
2. Open `index.html` in a browser (or deploy to GitHub Pages)
3. Get a free Groq API key at [console.groq.com](https://console.groq.com)
4. Paste your key in the app and start building your CV
